Projet scolaire

Le
contexte

Le site s’adapte à plusieurs résolutions d’écran

Ce projet a été réalisé dans le cadre du cours Design d’expérience et Conception Web IV. Le but était de développer un site événementiel, du design jusqu’à l’intégration HTML, CSS et JavaScript. Le site porte sur Musicon, un festival fictif de musique de jeux vidéos.

Le
design

Maquettes d’accueil du site version bureau

Le design a été réalisé en plusieurs étapes avec Figma. J’ai d’abord construit un guide de style et un moodboard pour la direction artistique du site. J’ai ensuite construit des wireframes pour établir l’emplacement des éléments du site. Finalement, j’ai complété les maquettes en fournissant une version mobile et bureau. Les maquettes inclues une version prototypée qui donne un aperçu des interactions possibles.

Les
integrations

Programmation de la composante Snackbar en JavaScript

Une fois le design terminé, j’ai développé le site web en HTML, CSS et JavaScript. J’ai ajusté le CSS pour que le site s’adapte peut importe l’appareil utilisé. Le site comporte plusieurs composantes JavaScript comme une snackbar utilisant le cache, une entête se cachant au défilement de la page et des carousels.
