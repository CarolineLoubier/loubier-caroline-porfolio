Le site s’adapte à plusieurs résolutions d’écran

Ce projet a été réalisé dans le cadre du cours Conception Web III et programmé en HTML, CSS et JavaScript. Le but était de programmer un site web sur le cinéma pouvait récupérer des données de l’API du site The Movie DB de façon dynamique.

Recuperer
le contenu

La classe MovieDB et ses paramètres sont utilisés pour récupérer les derniers films

Après avoir créé un compte TMDB, j'ai généré une clef API et défini plusieurs paramètres qui sont utilisés dans deux classes pour envoyer une requête des films voulus.

Afficher
le contenu

Méthode permettant d'afficher les derniers films

J'ai créé plusieurs méthodes bouclant sur les films voulus pour les afficher. Les méthodes utilisent un template créé en HTML et remplacent le contenu à l'intérieur de celui-ci par les données récupérées de TMDB.
